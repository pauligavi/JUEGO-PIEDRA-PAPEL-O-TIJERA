# JUEGO-PIEDRA-PAPEL-O-TIJERA
Este proyecto consiste en desarrollar un programa interactivo en Python que simula el clásico juego “Piedra, Papel o Tijera”. El usuario puede elegir una de las tres opciones disponibles, mientras que el programa genera automáticamente una elección aleatoria para la máquina. Luego, el sistema compara ambas elecciones utilizando estructuras lógicas para determinar si el usuario gana, pierde o empata.

El software incluye estructuras repetitivas que permiten jugar varias rondas mientras el usuario lo desee, además de validaciones para evitar errores en la entrada de datos. Todo el código está organizado en funciones para mejorar la claridad, la reutilización y el mantenimiento del programa
